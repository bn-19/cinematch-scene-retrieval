CineMatch

Describe a scene. Find your reference.

First search after the hosted Space wakes up can take 30-60 seconds while the model loads.

How it works

  1. 1
    Text query

    A scene description captures mood, composition, action, or lighting.

  2. 2
    CLIP embedding

    OpenCLIP maps the query into the same vector space as the film stills.

  3. 3
    Vector search

    ChromaDB compares cosine distance against the indexed scene collection.

  4. 4
    Scene matches

    The app returns the closest frames with metadata and similarity scores.
